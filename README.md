A website that bridges between a patient and specialists. A patient sends their symptoms to to a chat bot which will then convert the conversation into a medical file/history. Then later assigns a specific specialist to patients depending on the symptoms described. The specialist will then be able to have a conference call appointment with the patient to address the possible illness.
1. Home Page has all the information about Healthify and how it executes the assistance. Also includes contacts at the bottoms for any queries and feedbacks.
2. Nurse Pam chatbot: generates a medical report from the symptoms the user describes during the conversation, which will be used to refer the patient to the right specialist.
3. Services page is where the patient gets the help from their specialist which includes, video conferecing consultations, medical report viewing and appointment confirmations
4. Resources page is where the user can have acces to any tips and articles related to healthcare.
